## Discovery

Smithson Tennant discovered osmium in 1803 in the residue left when crude platinum was dissolved by aqua regia.

## Details
|Property       | value   | 
|---------------|--------:|
| Name          | Osmium  |
| Symbol        | OS      |
| Atomic number | 76      |
| Melting point | 3033 °C |
| Boiling point | 5012 °C |
